# shell-script skill

参照 ADR: [FLM_APP_0002: Shell スクリプトの基本ルール](../../../../vendor/flame/docs/adr/application/FLM_APP_0002__shell_script.md)

## 手順

ADR の規約に従ったうえで、本 skill が起動した工程では以下の procedural を完了させる。

- スクリプトの作成・更新後、実際に呼び出して動作確認する
  - 想定する代表的な引数 / 環境変数の組み合わせで 1 回以上実行する
  - 異常系 (引数欠落、依存コマンド不在等) を伴うスクリプトの場合はその経路も実行確認する
- 動作確認に用いた呼び出しコマンドと結果 (期待通りに終了したか、出力内容が想定通りか) を完了報告に明示する
- 静的検査 (shell lint) の通過は動作確認の代替にはならない (lint は構文・典型バグ検出に留まり、引数解釈や外部コマンド呼び出しの正しさは検出できない)
